4. Grid Layout

Свойства grid-контейнера:

  • display: grid - создает сетку
  • grid-template-columns - задает колонки
  • grid-template-rows - задает строки
  • gap - отступы между ячейками

Свойства grid-элементов:

  • grid-column - сколько колонок занимает
  • grid-row - сколько строк занимает
